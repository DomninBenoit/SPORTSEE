Project SportSee

SportSee is an application for the coaching

BACK-END

Download this repository [https://github.com/DomninBenoit/P12-front-end-dashboard.git] and follow the README.md

FRONT-END

Download this repository [https://github.com/DomninBenoit/SPORTSEE.git]

Prerequisite

• nodeJS (v14.18.1)
• yarn or npm

Dependencies

• create-react-app
• react-router-dom
• recharts

IDE

• VsCode

Launching the project

1. Fork the repository.
2. Clone it on your computer
3. The **yarn** or **npm** command will allow you to install the dependencies.
4. The **yarn start** or **npm start** command will allow you to run the front-end
